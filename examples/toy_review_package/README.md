# Unified Toy Review Package

This package is a synthetic toy dataset grouping metadata, human-confirmed document claims, images, tables, PV domain files, and raw PV measurements into a single unified directory for paper-level review verification testing.

## Contents
- metadata/: Paper DOI registry file
- documents/: Synthetic atomic claims for deterministic cross-document review
- images/: Mock images for exact duplication and perceptual similarity checks
- 	ables/: Mock source data tables for terminal digit and fixed delta reviews
- pv/: Mock PV metrics tables for PCE consistency and tandem checks
- 
aw_pv/: Mock raw J-V sweeps, EQE spectra, and Excel formula audit sheets
