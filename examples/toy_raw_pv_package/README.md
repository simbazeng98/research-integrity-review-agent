# Synthetic Toy Raw PV Package

All files in this directory are synthetic toy datasets designed to test the raw measurements recalculation layer (v0.11).
They do not represent any real paper, author, lab, or device.

## Structure
- `jv/`: Raw J–V sweep curves.
- `eqe/`: Raw EQE spectra.
- `excel/`: Data spreadsheets with formula audits.
- `reported/`: Reported summary metrics tables.
- `reference/`: Reference solar irradiance spectra.
