# Toy PV Package

All files in this folder are synthetic toy data designed to test the photovoltaics and materials domain consistency checks.
They do not represent any real paper, author, lab, or dataset.

## Contents
- `toy_pv_metrics_consistent.csv`: Consistent solar cell metrics (Voc, Jsc, FF, PCE).
- `toy_pv_metrics_inconsistent.csv`: Inconsistent solar cell metrics, FF percentage/fraction variations, non-standard light intensity.
- `toy_pv_eqe_jv_mismatch.tsv`: Current density comparisons with varying degrees of discrepancy (4%, 8%, 15%).
- `toy_pv_multisheet.xlsx`: Multi-sheet Excel workbook containing metrics, stability, and characterization data.
- `toy_pv_stability.md`: Stability test logs with some missing ISOS parameters.
- `toy_tandem_2t_4t.csv`: Tandem PV metrics illustrating 2T current mismatch and 4T efficiency summation scenarios.
- `toy_materials_characterization.csv`: Material characterization results with missing metadata.
